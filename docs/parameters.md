# Parameters Reference

| Parameter       | Type   | Description |
|-----------------|--------|-------------|
| line_spacing    | float  | Distance between lines (px). Controls density. |
| angles          | string | Comma-separated list of angles in degrees. |
| waveform        | enum   | sin, sin_b, saw, square. |
| modulation      | enum   | period, amplitude, both. |
| min_period      | float  | Minimum modulation period. |
| max_period      | float  | Maximum modulation period. |
| min_amplitude   | float  | Minimum modulation amplitude. |
| max_amplitude   | float  | Maximum modulation amplitude. |
| gamma           | float  | Luminance gamma correction. |
| line_width      | float  | Stroke width of generated lines. |
| units           | string | Units for stroke width. |
| remove          | bool   | Remove original image after shading. |
